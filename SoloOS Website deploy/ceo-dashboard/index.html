Questions

Before you buy.

Is this accounting software?

No. It is an organizational and planning system for understanding your business. It does not replace bookkeeping, tax filing, or advice from a qualified professional.

Where is my information stored?

Your entries save locally in the browser you use. The dashboard includes backup and restore so you can protect your work or move it to another device.

Do I need to install anything?

No. Download and unzip the files, then open the dashboard HTML file in a modern browser.

Will it calculate everything automatically?

It automatically calculates core totals, profit, margin, planning tax reserve, outstanding invoices, sales totals, average order value, pricing, break-even, and related figures from the information you enter. Some planning pages still use manual inputs.

Is it right for a brand-new business?

If you are not sure what you need yet, start with the free Business Health Scorecard. If you already have activity to track or want a system from day one, the Business Control Center is a strong starting point.
